إعدادات المحرر

أنشئ توكن محدود الصلاحية من GitHub Settings → Fine-grained Tokens

الإعدادات المطلوبة:

  • Repository access → Only select repositories → roko-tech.github.io
  • Permissions → Contents → Read and write
  • حدد مدة انتهاء قصيرة (30-90 يوم)

التوكن يُحفظ في المتصفح (sessionStorage) ولا يُرسل لأي طرف آخر.
